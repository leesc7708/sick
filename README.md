# 어디아파 (WhereHurts)

> 증상부터 처방까지, 원스톱 건강 가이드 앱

## 프로젝트 개요

일반인이 증상을 느꼈을 때 어떤 진료과를 가야 할지, 어떤 약을 먹어도 되는지 쉽게 알 수 있도록 도와주는 헬스케어 앱입니다.

## 핵심 기능

### 1. 증상 체커 🩺
- 자연어/음성으로 증상 입력
- AI 기반 증상 분석
- 예상 질환 및 진료과 추천
- 긴급도 판단

### 2. 병원 찾기 🏥
- 추천 진료과 기반 병원 검색
- 실시간 대기 현황
- 예상 진료비 안내
- 예약 연동

### 3. 약 상호작용 체크 💊
- 복용 중인 약 등록
- 약-약, 약-음식 상호작용 확인
- 일반의약품 추천
- 대체 약품 안내

### 4. 복약 알림 ⏰
- 시간별 복약 알림
- 복약 준수율 통계
- 가족 건강 관리

## 기술 스택 (예정)

- **Frontend**: React Native / Flutter
- **Backend**: Node.js / Python
- **AI**: OpenAI API / Claude API
- **Database**: PostgreSQL, Redis
- **APIs**: 식약처, 건강보험심사평가원, 네이버 지도
